# NetMarkdown

项目说明

## 建立工程

```sh
npm install
```
### 安装依赖

```sh
npm install vue-showdown@4.1.1
npm install naive-ui@2.34.4
npm install vfonts@0.0.3
npm install vue-router@4.2.4
```

### 编译和热加载dev

```sh
npm run dev
```
只调试到这里就可以了
-----

### Compile and Minify for Production

```sh
npm run build
```

### Run Unit Tests with [Vitest](https://vitest.dev/)

```sh
npm run test:unit
```
